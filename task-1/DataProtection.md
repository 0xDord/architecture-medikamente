|------------------|---------|--------|-----------|-----------|
| Медицинские карты | AES-256 | -      | для BI/ML | Критический |
| Результаты анализов | AES-256 | -      | для ML    | Критический |
| ФИО, паспорт | AES-256 | в логах | для аналитики | Критический |
| Телефон, email | AES-256 | в логах | для аналитики | Высокий |
| Адреса | AES-256 | частично | для аналитики | Высокий |
| Платёжные данные | PCI DSS | в UI  | агрегация | Критический |

## Механизм тегирования данных

#### Уровень конфиденциальности 
- **PUBLIC** — публичные данные
- **INTERNAL** — внутренние данные компании
- **CONFIDENTIAL** — конфиденциальные данные
- **RESTRICTED** — данные ограниченного доступа (медицинская тайна)
- **TOP_SECRET** — критически конфиденциальные данные

#### Категория данных
- **PII** — персональные данные (152-ФЗ)
- **PHI** — медицинские данные (врачебная тайна)
- **FIN** — финансовые данные
- **PCI** — платёжные данные (PCI DSS)
- **ADT** — аудиторские данные
